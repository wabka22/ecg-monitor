# ESP32 Dual Stream Manager

Программа на C# для управления двумя устройствами ESP32, поддерживающая параллельный стриминг данных через Wi-Fi. Позволяет настраивать ESP32 на домашнюю сеть, запускать стриминг данных и управлять несколькими устройствами одновременно.

## 🚀 Возможности

- **Двухэтапное подключение**: Подключение к AP-режиму ESP32, затем настройка на домашнюю Wi-Fi.
- **Параллельный стриминг**: Одновременное получение данных с двух ESP32 в отдельных потоках.
- **Поддержка различных типов сигналов**: 
  - Синусоидальный/косинусоидальный сигнал (для тестирования).
  - Сигнал ЭКГ с датчика AD8232 (реальные медицинские данные).
  - Любые другие аналоговые данные, передаваемые по TCP.
- **Управление конфигурацией**: Настройка сетей, IP-адресов и портов для каждого устройства.
- **Автоматическое обнаружение**: Поиск ESP32 в домашней сети.
- **Безопасность**: Остановка активных стримов перед новыми операциями.
- **Логирование**: Подробные логи с метками времени и уровнями.

## 📋 Требования

- **ОС**: Windows (из-за использования `netsh` для управления Wi-Fi).
- **.NET Framework**: 4.7.2 или выше.
- **ESP32**: С прошивкой, поддерживающей стриминг данных по TCP.

## 📦 Установка и запуск программы

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/wabka22/esp-32-wifi
   ```
2. Откройте проект в Visual Studio или используйте dotnet CLI для сборки.
3. Соберите и запустите приложение.
4. При первом запуске будет создан файл конфигурации config.json – при необходимости отредактируйте его под свои устройства.

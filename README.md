# ESP32 Dual Stream Manager

Программа на C# для управления двумя устройствами ESP32, поддерживающая параллельный стриминг данных (косинус и синус) через Wi-Fi. Позволяет настраивать ESP32 на домашнюю сеть, запускать стриминг данных и управлять несколькими устройствами одновременно.

## 🚀 Возможности

- **Двухэтапное подключение**: Подключение к AP-режиму ESP32, затем настройка на домашнюю Wi-Fi.
- **Параллельный стриминг**: Одновременное получение данных с двух ESP32 в отдельных потоках.
- **Управление конфигурацией**: Настройка сетей, IP-адресов и портов для каждого устройства.
- **Автоматическое обнаружение**: Поиск ESP32 в домашней сети.
- **Безопасность**: Остановка активных стримов перед новыми операциями.
- **Логирование**: Подробные логи с метками времени и уровнями.

## 📋 Требования

- **ОС**: Windows (из-за использования `netsh` для управления Wi-Fi).
- **.NET Framework**: 4.7.2 или выше.
- **ESP32**: С прошивкой для стриминга (косинус или синус), поддерживающей команды:
  - `SET` - настройка Wi-Fi.
  - `STATUS` - статус устройства.
  - `START_STREAM` - начать стриминг.
  - `STOP_STREAM` - остановить стриминг.
  - `CLEAR` - очистить настройки.
